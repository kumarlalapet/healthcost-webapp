Cost Difference for Medical Conditions / Treatments

The tool is a simple effort to provide the cost difference associated to a medical condition or treatment. The CMS Medicare Medicaid services department has released the 2012 Medicare Provider Utilization and Payment Data. This data set has close to 10 million payment records made by the government to the Medicare providers nation wide.

The tool utilizes the data and aggregates the payment data nation wide, state wide and city (using 3 map reduce jobs). The result data set is loaded into HIVE tables. The accompanying web app utilizes Hive Server 2 to load the data using HIVE JDBC driver. The web app is a rudimentary UI to search the aggregated data set.
